# Automotive Issue Analytics — Power BI Dashboard

An interactive **Power BI dashboard** that visualizes **global vehicle issues, defect patterns, and bug trends** across multiple automotive markets.  
This project helps identify quality gaps, track defect resolution, and provide **data-driven insights** for continuous improvement in automotive reliability.

---

## Overview

The **Automotive Issue Analytics Dashboard** consolidates vehicle issue and diagnostic data from global sources into one interactive Power BI interface.  
It enables stakeholders to analyze issue patterns, visualize regional performance, and monitor progress across product lines in real time.  

By leveraging **Azure Maps**, the dashboard overlays geographical trends, providing clarity on market-specific reliability concerns and helping teams focus on high-impact areas.

---

## Objectives

- Identify **defect frequency and distribution** by model, region, and issue type.  
- Track **root-cause analysis efficiency** and monitor defect resolution cycles.  
- Improve **data transparency** between engineering, validation, and quality teams.  
- Deliver **visual intelligence** to guide design, testing, and manufacturing decisions.  
- Strengthen product reliability and customer satisfaction through **predictive insights**.

---

## Key Features

### Global Vehicle Issue Tracking
Centralized visualization of defects and recurring issue types across Honda and Acura models, highlighting high-frequency components and critical failure zones.

### Root-Cause and Resolution Metrics
Visual indicators measure average resolution time, closure rate, and issue severity—resulting in a **35% faster root-cause identification** and **50% improvement** in resolution tracking.

### Region-wise Analytics with Azure Maps
Interactive map visuals pinpoint geographic defect clusters, enabling precise regional analysis and quality benchmarking across markets.

### Data Integration and Refresh
Built with **Excel**, **SQL**, and **Power Query** connections for seamless data refresh and large-scale analysis.

### DAX-Driven KPIs
Dynamic calculations measure key performance indicators such as defect density, issue recurrence, and average resolution time.

## How to Use

1. **Download or clone** the repository.  
2. Open the `.pbix` file in **Power BI Desktop**.  
3. Refresh data connections to sync with the latest Excel/SQL sources.  
4. Interact using filters and slicers by **region**, **model**, or **defect type**.  
5. Use **Azure Maps visuals** to explore global reliability distribution.  
6. Publish to **Power BI Service** for organization-wide access.
